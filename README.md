# Ventas App

Aplicación móvil para gestión de ventas, créditos y cobranzas.

## 🚀 Características

- Gestión de clientes
- Administración de créditos
- Registro de pagos con GPS
- Almacenamiento local SQLite
- Funciona 100% offline

## 📱 Descargar APK

Los APK compilados están disponibles en:

- [Releases](../../releases) - Versiones estables
- [Actions](../../actions) - Compilaciones automáticas

## 🛠️ Desarrollo

```bash
cd mobile
pip install kivy kivymd plyer
python main.py
```

## 📋 Requisitos

- Android 9.0 (API 28) o superior
- Permisos: Ubicación, Almacenamiento
